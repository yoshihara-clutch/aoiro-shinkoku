# かんたん青色申告サポート（業務委託美容師向け）

業務委託で働く美容師さんが、確定申告（青色申告・55万円控除）に向けて、日々の収入と経費をスマホでかんたんに記録し、年末に「科目別の集計」と「保存用の帳簿（CSV）」を書き出すためのWebアプリです。簿記の知識がなくても使えるよう、専門用語を避けています。

## 公開URL（スマホで開けます）

https://yoshihara-clutch.github.io/aoiro-shinkoku/

リンクをSafari／Chromeで開き、「ホーム画面に追加」するとアプリのように使えます。

## できること

- **収入の記録** … 日付・店名・金額・源泉徴収額（必須は金額だけ）
- **経費の記録** … やさしいカテゴリを選ぶと、内部で青色申告決算書の正式科目に自動対応
- **レシート読み取り（OCR）** … 撮影すると端末内で金額を読み取り自動入力（画像は保存・送信しません／読み取りは目安なので要確認）
- **自宅ぶんの計算（家事按分）** … 家賃・光熱費・通信費などを「仕事で使う割合（％）」で経費化
- **年間サマリー** … 総収入・科目別経費・差引所得・源泉徴収合計をひと目で
- **書き出し** … 科目別集計CSV／取引明細CSV（確定申告ソフトへの転記・保存用）

## 技術・プライバシー

- 単一のHTMLファイル1枚で完結（外部サーバー・DB・ビルド工程なし）
- HTML / CSS / 素のJavaScript のみ
- データはブラウザの **localStorage に保存され、端末の外には一切送信されません**
- OCRエンジン（Tesseract.js）のみ、初回利用時にCDNから読み込みます（レシート画像自体は送信しません）

## 大切な注意

- このアプリは「記録と集計」を助けるもので、確定申告書そのものは作りません。
- 税額・控除の最終判断は、税理士・税務署・国税庁e-Taxで確認してください。
- 青色申告には事前の「青色申告承認申請書」の提出が必要です。
- 端末の変更やブラウザのデータ削除で記録は消えます。こまめにCSV書き出しを。
